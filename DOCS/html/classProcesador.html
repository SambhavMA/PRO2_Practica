Detailed Description

Clase capaz de ejecutar y almacenar procesos.

Clase capaz de almacenar o eliminar procesos en sus unidades de memoria. También permite compactar estos procesos en la posiciones de almacenamiento-

◆ alta_proceso()

int Procesador::alta_proceso ( const Proceso &  p)

Se intenta dar de alta un proceso en el procesador.

Precondition
cierto.
Postcondition
Si hay un hueco de x posiciones de memoria donde x >= n (donde n son la memoria que ocupa el proceso p), el hueco es ocupado con esas n posiciones de memoria desde la poscion mas alta y se retorna 0. Si hay varios huecos que cumplen estas condiciones, se escoge el más ajustado con posición inicial de memoria más pequeña. Si en el procesador ya hay un proceso con el mismo identificador de p, se retorna 1. Si no hay hueco lo suficientemente grande se retorna 2 \coste logaritmico.

◆ compactar_memoria()

void Procesador::compactar_memoria ( )

Se compacta la memoria de un procesador.

Precondition
cierto.
Postcondition
Si hay procesos ejecutandose, se mueven las posiciones de memoria ocupadas por los mismos hacia el principio, de manera que no queden espacios vacíos entre cada proceso, éstos no se solapen y se respete el orden inicial. Devuelve cierto si existe procesador con id = id_proceso, en caso contrario vuelve falso. \coste lineal respecto al numero de procesos.
